# AirCloneDroid

Un clone de AirDroid, qui utilise [PAW Server](http://paw-android.fun2code.de/).

## Prérequis

- Un téléphone en 4+ (pas de support des anciennes API)
- Un IDE configuré pour le développement Android (Eclipse ou Android Studio)
